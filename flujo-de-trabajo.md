1. Al crear algo, ustedes crearan una nueva rama a partir de la rama main, y trabajaran ahi. Luego de que sus cambios sean aprobados, se hara el merge, es decir, llevar sus cambios a la rama principal. 

Recien clonen el repositorio, asegurense de escribir el comando:

~ git pull

Luego, para crear la rama a partir de la rama principal, escribiran el comando: 

~ git switch -c nombre-de-tu-nueva-rama

2. Cada vez que se cree una nueva view, componente, layout, etc, deben de importarlo con <script> en su correspondiente sesión en el archivo index.html. 

3. Al crear un nuevo componente, script o vista, deben de registrarla también en app.js, pueden referenciarse de lo que ya está hecho.
